# Black Force 007 — Cyber Quiz (Mobile-first)

Small, mobile-first quiz game with cyber/neon UI. Ready to host on GitHub Pages.

## How to run locally (mobile)
1. Put `index.html`, `style.css`, `script.js` in one folder.
2. Open `index.html` in mobile browser (Acode preview or file manager).
3. Play.

## Deploy to GitHub Pages (quick)
1. Create a new public repo on GitHub, e.g. `blackforce-quiz`.
2. Upload the files (`index.html`, `style.css`, `script.js`, `README.md`).
3. Settings → Pages → Source: `main` branch / root → Save.
4. After a minute open `https://<your-username>.github.io/blackforce-quiz/`.

## Notes
- Local leaderboard and player progress saved in `localStorage`.
- Add/modify questions inside `script.js` `QUESTIONS` array.
